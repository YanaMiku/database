# 🎵 Database Sound Effect - Drama Nyi Roro Kidul (XI TKJ 2)

Repository ini berisi kumpulan sound effect (SFX) dan musik latar untuk pementasan drama kolosal **"Legenda Nyi Roro Kidul"** karya XI TKJ 2.  
Seluruh file audio dikelompokkan berdasarkan babak (scene).

## 📁 Struktur Folder

- `/scene 1` - Hutan Rimba & Pernikahan Prabu Siliwangi
- `/scene 2` - Kerajaan Padjajaran & Intrik Istana
- `/scene 3` - Kutukan Sang Dukun & Pengusiran
- `/scene 4` - Perjalanan ke Selatan & Transformasi
